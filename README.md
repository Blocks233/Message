C++基于UDP的聊天软件 2nd ver2.31
====

输入框部分
-----
使用UDP想服务器发送数据<br>
数据长度限制2048字节(可在代码里调整)<br>

窗口部分
-----
使用udp向FRP客户端交换数据<br>
支持多输入框向一个窗口发送数据<br>

FRP部分
-----
服务器与客户端反向tcp连接<br>
请参照 https://github.com/fatedier/frp<br>

使用教程
-----
先使用计算MD5计算出密钥<br>
将id(即端口)与密钥输入窗口<br>
客户端安装提示输入数据<br>
请在程序关闭前不要关闭frp客户端<br>
